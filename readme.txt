API Documentation


1. Country
GET
 to get the country data you must pass the url as it is in the example below 
'http://localhost/php_demo/api/get_country.php?limit=5&search=GBP'
continent_code is the search column

POST
to import csv file use
'['server Address']/php_demo/api/post_country.php




2. Currency
GET
 to get the currency data you must pass the url as it is in the example below 
'['server Address']/php_demo/api/get_currency.php?limit=5&search=AS'
iso-code is the search column.

POST
to import csv file use
'['server Address']/php_demo/api/post_currencies.php
